# Buzz Bombers Game 🎮

This is a classic-style 2D shooter game created as my Programming Fundamentals (PF) project.  
Made using C++ and sfml

---

## 🛠️ Technologies Used
- C++
- Sfml
  
---

## 🎮 Features
- Multiple enemies with specific movement
- Bullet shooting
- Collision detection
- Score tracking
- Sound effects
- multiple levels
- leaderboard
- Menu
